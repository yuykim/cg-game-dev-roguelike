# Time Echo ASCII Levels

## Legend

- `.` empty
- `#` solid platform
- `B` breakable wall, 1 hit
- `X` reinforced breakable wall, 2 hits
- `=` moving platform
- `p` player or ghost button
- `g` ghost-only button
- `D` door, open while any button is pressed
- `S` spawn
- `E` exit
- `^` spike hazard
- `o` orb hazard
- `|` vertical laser
- `-` horizontal laser
