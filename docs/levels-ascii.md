# ASCII Level Drafts

## Legend

- `.` empty
- `#` solid platform
- `B` breakable wall, destroyed with `J`
- `=` moving platform
- `S` spawn
- `E` exit
- `^` spike hazard
- `o` orb hazard
- `|` vertical on/off laser
- `-` horizontal on/off laser
